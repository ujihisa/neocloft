## Usage

* `plugins/neocloft.jar`
* `plugins/neocloft/*.clj`
* `plugins/neocloft/*.jar`

optional

* `plugins/neocloft-jruby.jar`
* `plugins/neocloft-jruby/*.rb`

## Run

Download `clojure-1.5.1.jar` at your craftbukkit dir.

Copy `run` there and execute it.
